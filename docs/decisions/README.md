# Architecture Decision Records

This directory stores Architecture Decision Records (ADRs) for SPIRE.

Each ADR follows:

1. **Context**
2. **Decision**
3. **Consequences**

Current ADRs:

- `ADR-0001`: Stateless IPC boundary between frontend, Tauri bridge, and kernel
- `ADR-0002`: Dynamic widget registry as canonical render routing
- `ADR-0003`: Dual layout memory for canvas and docking modes
- `ADR-0004`: DAG-based pipeline execution with stale-propagation caches
- `ADR-0005`: Backend transport abstraction with runtime schema validation
