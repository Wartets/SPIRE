# Rust API (Generated)

This page tracks the generated Rust API documentation for SPIRE crates.

## Local Generation

Generate docs from the repository root:

- `cargo doc --workspace --no-deps`

The generated output is written under `target/doc`.

## Published Paths

In documentation deployment builds, generated Rust API pages are expected under:

- `/api` (site-level route)

## Covered Crates

- `spire-kernel`
- `spire-bindings`
- `spire-desktop-backend`
